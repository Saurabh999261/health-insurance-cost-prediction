# 🏥 Health Insurance Cost Prediction

## 📌 Project Overview
Predicted health insurance medical charges using Machine Learning techniques.
Performed end-to-end Data Analysis on 1,300+ insurance records to identify
key cost drivers and built predictive models using Python and Scikit-learn.

## 🎯 Objective
To identify factors affecting health insurance costs and build a regression
model that accurately predicts medical charges for new customers.

## 🔍 Key Findings
- Identified 3 major cost drivers — BMI, smoking status, and age
- Smokers pay on average 3-4x more than non-smokers
- BMI above 30 significantly increases medical charges

## 🛠️ Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## ⚙️ Models Used
| Model | R² Score |
|---|---|
| Linear Regression | Baseline model |
| Random Forest | Best performer |

## 📊 Project Steps
1. Data loading and exploration
2. Exploratory Data Analysis (EDA)
3. Data preprocessing and cleaning
4. Feature engineering
5. Model building and comparison
6. Model evaluation using R² and RMSE

## 📁 Dataset
- Records: 1,300+
- Features: Age, BMI, Smoking Status,
  Region, Number of Children, Charges
